# Chronos Launcher

A modern, minimal Android launcher built with Jetpack Compose.

## Features

- **Minimalist Design**: Focused on productivity and reducing distractions.
- **Media Integration**: Built-in media controls via notification listener.
- **Quick Access**: Fast access to your most-used apps and information.
- **Jetpack Compose**: Built entirely using modern Android UI toolkit.

## Permissions

This app requires several permissions to function as a full-featured launcher:
- `QUERY_ALL_PACKAGES`: To list and launch installed apps.
- `BIND_NOTIFICATION_LISTENER_SERVICE`: To provide media control integration.
- `ACCESS_COARSE_LOCATION`: For weather-related features.

## Getting Started

### Prerequisites

- Android Studio Hedgehog (2023.1.1) or newer
- Android SDK 34
- JDK 17

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/chronos-launcher.git
   ```
2. Open the project in Android Studio.
3. Build and run on your device or emulator.
